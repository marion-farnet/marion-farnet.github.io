DRAWING 1, DRAWING 2, & FIGURE:

2D CLASSWORK

KSU's 2D class is one of the first foundation courses students must take.
In this class they are given specific guidelines which they cannot break
and learn how to create compositions out of basic shapes in order to learn elements of design.

TYPE AND DESIGN CLASSWORK:

Type and Design uses text and color to create successful compositions.
Students are provided text and with the text have to follow certain
guidlines of design, which they later get to break.
